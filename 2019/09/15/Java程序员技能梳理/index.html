Java程序员技能梳理
发表于 2019-09-15| 更新于 2019-11-02

前言

Java后端开发需要学习的东西很多，有些东西需要去深入了解，知道源码，知道异常情况处理，知道底层实现等等，经常会有想学习，但不知道从何下手的情况。本文作为博客的开篇，想先简要阐述一下本人的学习计划安排（只是个人的理解，可能适合自己，但不一定适合读者的情况）。

笔者是双非硕士毕业，目前工作半年，校招并未拿到心仪的offer，现在工作了半年了，回过头来静想一下走过的校招之路，感觉走了很多弯路，不知道面试是啥，笔试是啥，有什么该注意的，最重要的是不知道该学啥。现在工作了半年，也明白了Java程序员需要的”内功”和”外功”，因此想写这篇博文，记录一下需要学习的内容，也作为后面学习的指导。其实不论是校招还是社招，只要是做Java开发，那么本文的学习内容骨干都是适合的。

Java程序员的必备技能

笔者认为，对于校招生或刚参加工作的同学，基础还是最重要的，上图是笔者认为需要学习的一些主要部分，别的东西知道更好，不知道也ok吧，但上图内的东西都是最主要的和面试经常会问到的内容。个人认为，主要分为以下几个部分：

  1. 算法和数据结构
  2. 编程语言（Java）
  3. spring、mybatis框架
  4. 分布式
  5. 数据库

上图中，写了一下大体需要知道和掌握的主要知识点，和学习的方式，可以参照着学习。最后想到以前看到一句话：在刚开始追求完美的，往往最后都是失败告终的。现在回过头来想这句话，真的很有道理，刚开始学jdk源码，笔者还想着对几乎所有主要的类都彻底研究透了源码，在写了10几篇博客之后发现，这样太浪费时间了…还是应该把最主要的部分，例如集合、多线程等先了解了在去看细枝末节的东西，这个要在以后的学习中注意。
